\# AI Resume Analyzer (Offline NLP)



This project is an AI-based Resume Analyzer built using Natural Language Processing techniques.  

It analyzes resume text, extracts important keywords, detects skills, scores the resume, and provides improvement suggestions.



\## Features

\- Resume scoring

\- Skill detection

\- Keyword extraction using TF-IDF

\- Improvement suggestions

\- Works completely offline (no API or billing required)



\## Tech Stack

\- Python

\- Streamlit

\- NLTK

\- Scikit-learn
